**yolo v8-LC-CBAM model **
________________________________
This project uses the YOLOv8s model to accurately detect defects in insulators 
and generates an EXE file that can be run on a computer. 
Since the EXE file is large, it must be downloaded. Download link:https://pan.baidu.com/s/1pNtOW3dqtLiDDR8I_3MxRQ?pwd=cfsf
